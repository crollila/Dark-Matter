Environment Sprite Inspector

Inspect every env_* sheet as a grid (default 8×8). The overlay grid and the drawn image share the exact same scale, so cell boxes line up with the art. Click a cell for its col,row, a copy-ready { col, row }, and ready-made ENV_TERRAIN_ROLES / ENV_OBJECT_ROLES lines for js/sprites.js. Terrain roles (green) are real ground/wall/liquid tiles; object roles (orange) are props drawn on top. The alpha-trim bounds overlay (green box = trimmed, red = full cell: opaque or a tainted/file:// canvas) shows the exact visible art the game crops; the big preview renders that crop in the chosen mode (terrain = cover/fill the tile, object = contain + bottom-center). This explains why an untrimmed cell looked small/off-centre. Use cols/rows if a sheet isn't 8×8 and alpha thr to tune the transparency cutoff. This page never changes gameplay.
